## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
# or
$ pnpm install
# or
$ yarn install
```

## Running the app

```bash
# development
$ npm run dev

# production mode
$ npm run build
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:coverage

# e2e tests
$ npm run test:e2e
```

## KitiumAI Packages

This project includes the following KitiumAI packages:

- `@kitiumai/utils-ts` - TypeScript utilities
- `@kitiumai/logger` - Logging utilities
- `@kitiumai/types` - Shared TypeScript types
- `@kitiumai/config` - Shared configuration
- `@kitiumai/lint` - ESLint configuration
- `@kitiumai/test-core` - Testing utilities
- `@kitiumai/vitest-helpers` - Vitest helpers
