# {{PROJECT_NAME}}

A React + TypeScript project powered by KitiumAI.

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Run tests
pnpm test
```

## Scripts

- `pnpm dev` - Start development server on port 3000
- `pnpm build` - Build for production
- `pnpm preview` - Preview production build
- `pnpm test` - Run tests
- `pnpm lint` - Lint the codebase
- `pnpm format` - Format the codebase

## KitiumAI Packages

This project includes:
- `@kitiumai/utils-react` - React hooks and utilities
- `@kitiumai/logger` - Logging utilities
- `@kitiumai/types` - TypeScript type definitions
- `@kitiumai/config` - Project configuration
- `@kitiumai/lint` - Linting configuration
- `@kitiumai/test-core` - Testing utilities
- `@kitiumai/jest-helpers` - Jest testing helpers

## Tech Stack

- React 18
- TypeScript 5
- Vite 5
- Jest for testing
