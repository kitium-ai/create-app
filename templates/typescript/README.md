A TypeScript project powered by KitiumAI.

## Getting Started

```bash
# Install dependencies
pnpm install

# Run in development mode
pnpm dev

# Build for production
pnpm build

# Run tests
pnpm test
```

## Scripts

- `pnpm dev` - Run the project in watch mode
- `pnpm build` - Build the project
- `pnpm test` - Run tests
- `pnpm lint` - Lint the codebase
- `pnpm format` - Format the codebase
